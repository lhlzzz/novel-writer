# 第003章 她怀孕六周，我丢了三个月

五十万到账后的第三分钟，刘成的电话打了进来。

林川站在城南立交桥下，手里还攥着那颗被朵朵塞过来的糖。糖纸已经被他捏得发皱，掌心里却还残着一点孩子的体温。手机屏幕上，一边是银行到账短信，一边是那个陌生号码发来的三行字。

“今晚八点，老地方见。”

“我怀孕了。”

“孩子是你的。”

来电显示没有名字，只有一串号码。可林川几乎不用猜也知道是谁。

他没有接。

电话响到第六声，自动挂断。下一秒，刘成发来一条短信，字很短，像他本人站在屏幕后面，慢慢把判词念出来。

“钱别急着还。账不喜欢插队。”

林川后背一点点冷下来。

他刚才还在想，这五十万是台阶。现在才明白，它更像账递给他的一把刀，刀柄朝着他，刀刃朝着身边所有还没被删干净的人。

老马抱着朵朵站在几步外，女儿还在小口喘气。他想问林川怎么了，嘴唇动了动，最终没开口。经过刚才那一遭，老马大概也知道，有些东西不是多问就能问出答案的。

林川低头点开通讯录。

最底下那个联系人还在。

苏。

只有一个字。

他盯着它看了两秒，手指悬在屏幕上方，没敢碰。那张账单还像冰片一样卡在脑子里：苏念通讯录字段轻度损毁。轻度。账把一个人的名字擦掉一半，也只叫轻度。

如果他现在点开，会不会把剩下那个“苏”也擦掉？

如果他回复一句“你是谁”，会不会连她发来的那三句话都变成空白？

五十万躺在账户里，像一笔刚到账的赃款。林川很清楚，拿它去还刘成，只能让催收单上少一截数字；拿它去查这个号码、查医院记录、查白昼科技旧账，才有可能把被账收走的人抢回来。

他打开邮箱，翻出白昼科技当年绑定的云服务账号。

账号早就欠费停用，红色提示挂在页面最上面：

“数据归档冻结。续费后可恢复最近一次备份索引。”

欠费金额：32800 元。

林川盯着那个数字，忽然笑了一下。

以前三万多能让他跟王海吵一整夜，王海会骂他“你他妈把服务器当祖宗供着”，苏念会坐在旁边，一笔一笔算这笔钱到底能撑几天。现在他连犹豫都没有，直接点了支付。

付款成功的提示跳出来时，账户余额少了一截。

五十万第一次变成了东西。

不是还债。

是把一段快被清掉的旧生活，从账的手里硬买回来几分钟。

备份索引恢复得很慢。页面转了十几秒，吐出三条残缺记录：

0512 异常复盘。

南岸咖啡。

苏念：如果你哪天又把我忘了，就来这里。

林川的心脏像被人攥住。

三个月前，苏念不是突然从他生活里消失的。

是他先从她那里开始坏掉。

晚上七点四十九，林川站在南岸咖啡门口。

这家店开在老城区拐角，门口风铃还是旧式铜铃，风一吹，声音很轻，像某种不敢落地的提醒。玻璃门上贴着半透明的雨痕贴纸，角落有一块旧牌子，写着“老座位已预留”。林川看见那几个字，太阳穴忽然一跳。

他应该来过这里。

不止一次。

可脑子里没有画面。

只有一种很荒唐的身体记忆：他下意识摸了摸右边口袋，像那里本该放一把伞；又下意识看向靠窗最里面的位置，像那里一直有人等他。

风铃响了一声。

林川推门进去。

店里没几个人。最里面的卡座坐着一个女人，米白色风衣，头发挽在脑后，手边放着一杯几乎没动过的热美式。她面前有一张 B 超单，一支黑色签字笔，还有一个已经拆开的牛皮纸袋。

她没有抬头。

林川站在过道里，忽然说不出话。

因为通讯录里剩下的那个“苏”字，在看见她的一瞬间开始发烫。他知道这个人很重要，重要到身体比脑子先认出来。可他叫不出她的完整名字。

女人终于抬眼。

她看见林川，目光先停在他的脸上，又落到他攥着旧钢笔的手上。那一瞬间，她的眼神很轻地动了一下，像是有某句话到了嘴边，又被她咽回去。

“你迟到了九分钟。”她说。

声音很冷静。

冷静得像早就演练过很多遍，今天无论他来不来，她都不会让自己先崩。

林川喉咙发干。

“苏……”

他只说出这一个字。

女人的手指在咖啡杯边停住。

她没有立刻发火，也没有哭。她只是看着他，几秒钟后，把杯子往旁边推开。那个动作很小，却像把他们之间最后一点温度也推远了。

“苏念。”她替他说完，“林川，你连我的名字都只剩一个字了？”

这句话没有抬高声音，却比吼出来更疼。

林川张了张嘴，最后只说：“对不起。”

苏念笑了一下。

那不是笑，是一种被气到极点后的平静。她低头摸了摸左手腕，那里什么都没有，却留下了一圈比皮肤更浅的印子，像曾经戴过什么东西。

“你以前最会说这三个字。”她说，“项目延期，你说对不起；钱不够，你说对不起；王海跟你吵翻，你也说对不起。可你每次说完，第二天还是照样往前扛，照样替所有人做决定。”

王海两个字落下时，林川心口猛地一紧。

他下意识抬头：“你记得王海？”

苏念的眼神变了。

那一点变化很快，快到普通人可能会错过。可林川看见了。她不是疑惑，不是陌生，而是一瞬间的警惕。

“你也记得？”她问。

一个“也”字，让林川指尖发凉。

苏念没有继续追问。她把桌上的 B 超单推过来，纸边压在林川面前，动作克制得像在递一份合同。

“先看这个。”

林川低头。

姓名栏：苏念。

孕周：6 周 + 2 天。

检查时间：今天上午 10:41。

诊断建议那一栏很普通，普通到让人心里发沉。可父亲信息栏那里，却不是空白，也不是“未填”。

它打印着三个字：不存在。

林川盯着那三个字，呼吸一点点慢下来。

不是“未知”。

不是“未婚”。

是不存在。

苏念看着他的反应，眼底那点强撑出来的平静终于裂了一道缝。

“我上午去医院建档。”她说，“护士让我填父亲信息。我填了你的名字，系统退回来。填身份证号，也退回来。最后机器自己打印出这三个字。”

她顿了顿，指尖压在纸边，压得很紧。

“林川，我今天叫你来，不是为了让你负责。”

她抬头看他。

“我是想确认，你到底是人不见了，还是从这个世界里开始不见了。”

林川说不出话。

咖啡馆里的空调很足，可他后背全是汗。这两天的所有线，在这一刻突然连起来：王海查无此人，苏念名字少了一半，孩子父亲字段不存在。账不是在吓他。

账是在一点点把旧项目的人从现实里拆出来。

先拆王海。

再拆苏念。

最后轮到这个还没出生的孩子。

“你短信里说三个月。”林川强迫自己开口，“可 B 超单是六周。”

苏念看着他，像终于等到他问出一个像样的问题。

“三个月前，不是我怀孕。”她说，“是你第一次不认识我。”

林川的手指收紧。

苏念垂下眼，声音依旧稳，只是稳得让人更难受。

“三个月前，我们约在这里谈离婚后最后一笔账。你坐在现在这个位置，看了我很久，然后问我——‘我们以前认识吗？’”

她说到这里，停了一下。

“我那时候以为你在装。后来你又恢复了，跟我道歉，说自己太累，头疼，记忆断片。再后来，每隔一段时间，你就会忘掉一点。先是忘了我换过工作，忘了我们离婚那天为什么没吵架，忘了这家店为什么叫老地方。”

她抬手摸了一下手腕那圈浅痕。

“六周前，你还记得我。至少那天晚上，你记得。”

林川胸口像被什么东西狠狠撞了一下。

他应该有记忆。

他应该记得那天晚上怎么下雨，怎么进门，苏念有没有哭，他们有没有吵，旧钢笔是不是就在桌上。可他什么都抓不到。只有一个空洞的确定感：这件事发生过，而账正在把它从他身体里搬走。

林川把手机放到桌上。

通讯录界面还停在那个孤零零的“苏”字。

“我没有装。”他说，“我是真的在忘。”

苏念没有立刻相信他。

她只是看着那个“苏”字，看了很久。然后她伸手，在屏幕上点了一下。

联系人详情弹开。

名字栏里，原本只剩下“苏”。可在她手指碰到屏幕的瞬间，后面的“念”字像水底浮上来的影子，短暂地亮了一下。

苏念。

不到半秒，又散了。

只剩“苏”。

苏念的脸色终于变了。

她不是第一次知道异常存在。

林川从她的表情里看出来了。

“你见过这种事。”他说。

苏念收回手，指尖发白。

“我见过比这更糟的。”

“和白昼科技有关？”

苏念抬眼，眼神一下冷下来。

“你现在才想问白昼？”

这句话像一根针，扎得林川无处可躲。

他把那五十万到账后的付款记录调出来，推到苏念面前。

“我用钱恢复了白昼的云备份索引，只恢复出三条：0512 异常复盘、南岸咖啡，还有你那句‘如果你哪天又把我忘了，就来这里’。”

苏念的呼吸停了一瞬。

“你恢复了备份？”

“只恢复索引。”林川说，“更多内容被冻住了。我需要权限，或者原始凭证。”

苏念盯着他，眼底的怒意没有消失，但多了一点别的东西。不是原谅，是判断。她在重新判断眼前这个人到底是烂人、病人，还是某个她早就害怕会出现的结果。

“你终于舍得拿钱查事了。”她说。

林川没有反驳。

他知道这句话不只是讽刺。过去的他太擅长把钱投进一个又一个“再扛一轮”里，却不肯停下来查清楚自己到底把谁拖进了坑。

“我现在查。”他说。

苏念把牛皮纸袋推到他面前。

里面不是照片，也不是情书。

是一张旧门禁卡，一枚U盘，还有半张被撕下来的便签。便签边缘被水泡过，字迹有点花，但还能看清一行字：

0512 之后，别让林川一个人记。

落款没有名字。

只有一个被划掉又写回来的“王”字。

林川盯着那个字，脑子里像有一根线猛地绷紧。

王海。

他几乎脱口而出。

下一秒，手机屏幕自己暗了一下。

不是没电。

是那条通讯录里的“苏”字旁边，浮出了一行极淡的冷白字。

【关系债目触发】

对象：苏念 / 未出生子项。

异常类型：父亲字段不存在。

可执行校准：导入 0512 备份片段。

预期收益：恢复父亲字段，锁定孩子存在证明。

预期扣除：王海残余节点清零。

林川的血几乎凉透。

苏念看不见那几行字，但她看得见他的脸色。

“怎么了？”她问。

林川抬头看她。

他终于明白账为什么让他在第一个 bug 里看见王海查无此人，又在第二个 bug 里擦掉苏念的名字。

它不是随机收取代价。

它是在逼他用一个旧项目里还没被删干净的人，去换另一个人的存在证明。

“如果我查这枚 U 盘，”林川声音很低，“可能会彻底弄丢王海。”

苏念的眼神剧烈晃了一下。

很短。

可足够了。

她记得王海。

她不仅记得，她也怕失去他。

咖啡馆门口的风铃忽然响了。

林川和苏念同时看过去。

一个穿黑色外套的瘦子站在玻璃门外，嘴角有颗痣，手里拿着手机，正隔着玻璃看向他们这一桌。

城南立交桥后那辆黑色面包车副驾上的人。

苏念把B超单和U盘同时按住，声音压得很低。

“你被跟踪了。”

林川没有动。

手机上的冷白字还在继续往下掉。

【选择生成】

A：立即导入 0512 备份片段。

结果：恢复孩子父亲字段；王海残余节点清零。

B：临时冻结关系债目。

结果：保留王海残余节点；苏念字段继续损毁；倒计时 23:59:59。

冻结押金：当前账户余额全部。

林川看着“冻结押金”那一行，忽然笑了一下。

五十万刚到手，还没捂热。

账已经替它找好了去处。

苏念看着他：“林川？”

这一次，他没有说“我会想办法”。

他把手机转过去，让屏幕朝下，另一只手按住那枚 U 盘。

“我先不导入。”他说。

苏念的指尖顿住。

“为什么？”

“因为如果孩子是真的，王海也是真的。”林川抬起眼，声音发哑，却终于稳住了，“我不能用一个人，去证明另一个人存在。”

门外，瘦子已经推门。

风铃响起。

林川按下了 B。

银行卡扣款提示几乎同时弹出。

支出 467200 元。

备注：关系债目冻结押金。

手背上的“7”猛地一烫，数字边缘裂开一道细细的白痕。通讯录里那个“苏”字闪了一下，没恢复，也没继续消失。桌上的 B 超单父亲栏里，“不存在”三个字被一层灰色横线暂时压住，像一口还没盖死的棺材。

而门口的瘦子停下脚步，忽然笑了。

“林川。”他说，“刘哥让我来收一样东西。”

他的目光落在那枚 U 盘上。

“不是钱。”

“是还记得王海的人。”
