# 第001章 账给了我七天

凌晨三点，门外有人喊林川的全名。

不是电话，也不是短信。是砸门。出租屋那扇薄木门被砸得发抖，门框上的灰簌簌往下掉，茶几上半杯冷咖啡晃出一圈黑色涟漪，像一口小井被人从底下敲醒。

“林川。”

门外男人声音不高，却把整条楼道都压静了。

“三百万到期了。今晚十二点以前，你要么还钱，要么跟我走。”

林川坐在沙发边，手撑着额头，第一反应不是害怕，是羞耻。对门猫眼暗了一下，又亮起来；隔壁小夫妻的门链轻轻响了一声；楼下那个总在门口抽烟的外卖员大概也听见了，烟味顺着楼道往上浮，停在他门口，像有人提前给他的狼狈盖了个章。明天早上，他们也许不会当面说什么，可经过三楼时，会下意识压低声音，用余光扫他的门——哦，原来这屋住的是欠钱不还的人。

林川低头看了一眼自己的拖鞋。鞋面裂了，脚趾头顶着一块起皮的皮革。他忽然很想把脚往茶几底下缩，又觉得可笑：门外的人都堵到三楼了，他还在管一双破拖鞋会不会被邻居看见。

门外又砸了一下。墙角那只装旧资料的纸箱被震歪，露出半截褪色工牌。工牌上印着“白昼科技”，下面是三年前的林川：白衬衫，短头发，眼神亮得很欠揍，像真以为自己能把世界改一改。

现在他衬衫皱到发硬，两天没刮胡子，脚边堆着一摞催收函。最上面那张右上角露出一串编号：BD-0512-RE。电脑旁边的创业纪念牌早就不亮了，旧钢笔躺在键盘旁，笔帽缺了一块漆，是苏念送他的。他曾经穷到拿去二手店问价，对方看了一眼，说：“二十，不能再多。”

他没卖。不是因为舍得，是因为那一刻他突然觉得，如果连这支笔都卖掉，自己就真的一点过去都没有了。

“别装死。”门外的刘成说，“账不会等你睡醒。”

刘成是信诚催收的组长，四十来岁，讲话从不急，甚至很少骂人。他可怕就可怕在这里——不像讨债的，更像宣判的。每句话都慢，每个字都稳，稳得像已经替你算好了最后一口气值多少钱。

林川低头看手机。十七个未接来电挤在屏幕上，一半是信诚，一半是刘成。最下面还有一条昨晚没点开的短信：最后一次提醒，清理费用待结算。

清理费用。

他盯着那四个字，眼皮跳了一下。三百万明明是白昼科技倒掉以后他签下的旧债。那年办公室租金、服务器费用、员工补偿、项目违约，所有窟窿像一张张嘴，最后全都朝他张开。他不肯承认失败，对苏念说再扛一轮，对王海说下个版本一定能交，对债主说月底肯定还。说到最后，连他自己都听不出哪句是真，哪句是硬撑。

王海那时候靠在办公室门口，嘴里叼着没点燃的烟，骂他：“你他妈要真借，就别半路死。欠债可以，丢下烂摊子不行。”

苏念没骂他，只把水杯往他手边推了推。杯把朝着他右手边，是她照顾人时最不声张的习惯。

林川那时说：“放心，我能扛。”

可那句“能扛”，最后只剩下门外的砸门声、脚边的 BD-0512-RE、和一支二十块都不值的旧钢笔。

门外安静了两秒。林川知道，那不是他们走了，是刘成没耐心了。果然，下一秒，刘成的声音贴着门板响起：“我数三下。三下不开，我让人拆门。”

“一。”

林川站起来，脚碰到地上的充电线，差点绊倒。他扶住茶几，掌心碰到冷咖啡杯，冰得一激灵。

“二。”

他低头看了眼自己衬衫。领口皱着，袖口有咖啡渍。他下意识扣好最上面的扣子，又把袖口拉平。动作很可笑，可人越狼狈，越会死死抓住一点没用的体面。

“三——”

门开了。

楼道里的冷风、烟味、潮湿墙皮味，还有夜里站久了的人身上那股压不住的汗味，一起扑到脸上。刘成站在最前面，黑色风衣扣得严严实实，手里拿着打印好的催收函。身后两个年轻人一左一右堵住楼梯口，再后面站着个穿蓝色工装的中年男人，手里夹着欠款单，鞋尖沾泥，眼神躲躲闪闪，像不太敢看他。

那人林川见过两次。信诚底下跑腿的，姓马。平时不怎么说话，刘成让他递单，他就递单；让他守门，他就守门。昨晚在楼道阴影里，他像一块旧抹布，谁都能拿来擦一下脏水。

刘成看见林川，先扫了一眼他的领口。

“你还真是老样子。”他说，“越难堪，扣子扣得越紧。”

林川没接话。他知道刘成这种人。你越解释，他越像听笑话。真要动手，他前面会先给你一口温水，再把你整个人摁进冰里。

“三百万。”刘成把催收函递到他胸前，纸边抵住衬衫，“最后期限，今晚十二点。你要是还有办法，现在说。别一会儿哭，一会儿装硬气，浪费大家时间。”

林川低头看那张纸，没有接。催收函右上角还是那串编号：BD-0512-RE。编号下面，有一行淡得几乎看不清的灰字：清理费用，待结算。

“跟我们走一趟。”刘成说，“老板要见你。”

“见我干什么？”

“你欠的不是小钱，老板也不是有耐心的人。”刘成看着他，“看在你以前也算个体面人，我让你自己走。”

林川嘴角动了一下，想笑，没笑出来。

他确实没办法。车卖了，电脑卖了，能借的人早就被他借怕了。连苏念那支钢笔，他都拿去问过价。可三百万不是一堵墙，是一口井。他越往里扔东西，越听不见回声。

“给我三天。”他说。

刘成看了他一会儿，像听见一个小孩说明天能把天补上。

“三天？”他把这两个字咬了一遍，“林川，你打算去抢银行？”

“我会想办法。”

“这句话你说了三年。”刘成把催收函往前送，纸角顶得他胸口一疼，“看着我。我不是来听你想办法的，我是来收账的。”

林川终于伸手接过那张纸。纸很轻，轻得像一片薄薄的雪，可指尖碰到纸面的瞬间，他整个人却像被一块铁压住。最后一页是他的签名，歪歪扭扭，和三年前一样难看。他下意识想把那一角抚平，手指碰上去才发现纸角已经被刘成抵皱了，皱痕正好压过“林川”两个字。

三年前歪到今天。

刘成抬了抬手。身后两个年轻人同时往前半步，鞋底在水泥地上磨出轻响。老马也跟着动了一下，又马上停住，欠款单被他捏出一道皱。那声音很小，却像刀刃刮过骨头。

林川知道，下一秒他们就会把他带走。

可就在这时候，催收函上的日期跳了一下。

纸面最上方那行“到期日：今日二十四时”忽然褪成灰色，接着重新浮出来，变成“到期日：昨日二十四时”。下一秒，它又跳回“今日二十四时”。短短两秒，三个日期像坏掉的钟摆一样来回切换，连刘成按在纸边的手指都被那片灰白光影切成了几截。

楼道尽头那只电子钟也抽了一下。

03:17。

03:16。

03:17。

时间像被什么东西拽着脖子，硬生生往回拖了一秒。

林川猛地抬头。楼道停了。风停了，声控灯停了，刘成抬起的手停在半空，两个年轻人的鞋尖也停在原地。老马嘴唇微张，像正要说一句什么，却被冻在喉咙里。墙上那盏忽明忽暗的灯亮着，不再闪，灯丝像一截被冻住的火。

林川伸手碰了一下刘成的手臂。冰凉，不像人，像一块刚从冰柜里拿出来的肉。

“……我疯了？”他听见自己低声说。

下一秒，催收函边缘浮出冷白色的字。它不像印在纸上，也不像投在空气里，更像有人用刀尖在玻璃上刻出来的判词：异常检出，对象为三百万债务承诺链，类型为局部时间循环，触发人为林川，剩余寿命为 0 天。

林川的呼吸一下停住。

0 天。

这两个字比“异常”本身更荒唐，也更要命。他前一分钟还在担心明天邻居会不会拍下他被拖走的照片；下一分钟，一张不知道从哪里冒出来的账单告诉他，不是还不上钱这么简单——他连明天都没有了。

纸面继续往下渗字。待处理债目下面列着一句重复承诺：再给我一点时间。累计次数 47。判定为无效承诺污染现实时间流。可执行修复只有一项：终止拖延，确认失败。

林川盯着“确认失败”四个字，喉咙像被堵住。他想骂一句开什么玩笑，可骂不出来，因为那行字太准了。

三年前，他对苏念说再扛一轮；对王海说再给我一次机会；对投资人说下个版本一定能交付；对债主说月底肯定还；对自己说只要撑过去就好了。他以前总觉得，只要把话说稳，把扣子扣好，把“再扛一轮”说得像真的，别人就会再信他一次。

可催收函上的签名歪在那里，三年前歪到今天。林川忽然发现，自己这三年不是在扛，是在拖着所有人一起往井里沉。

原来连后悔都要收滞纳金。

他忽然笑了一下。那笑很轻，不像轻松，更像被逼到墙角后终于认栽。他抬头看冻在半空里的刘成，看那个缩着肩膀的老马，看自己手里那张薄得过分的催收函，声音低得几乎被楼道吞掉。

“这笔债，我认。”

没有反应。

他又说：“白昼科技是我撑死的。钱是我签的，王海和苏念是我拖下水的。我不是在想办法，我是在拖命。”

最后两个字出口时，胸口像被人狠狠撕开一道口子。他一直觉得男人不该这么承认。可现在反正没人听见，除了这一张看不懂来路的账单。

林川握紧那张纸，指节一点点发白。

“我扛不动了。”

楼道里那片静止的冷光忽然震了一下。催收函上的日期不再跳动，灰白色的字一行行坠下来，像判决落锤：局部时间循环被终止；本次收益，寿命加七天；三百万债务进入七日展期；本次扣除，白昼科技最后一夜，苏念递来那杯水时的温度。

林川脑子里像有什么东西被橡皮擦过。

他还记得那天办公室停电，记得王海骂骂咧咧去楼下买蜡烛，记得苏念把水杯推到他手边，记得杯口有一圈白雾。可他忽然记不起那杯水到底是热的，还是凉的。他甚至还记得杯把朝着右手边，却想不起自己接过来时，指腹碰到的是暖意，还是冷掉的瓷。

更糟的是，他刚刚才想起过苏念推杯子时指尖有没有被烫红。现在那一点细节像被擦掉，只剩一块发白的空。照片还在，人也还在，可他知道有一小块一起活过的证据永远没了。

很小的一点记忆。小到换七天命，听起来甚至很划算。可林川在那一瞬间出了一身冷汗。

如果第一次只是温度，第二次呢？第三次呢？等他把三百万补上，等他把这些所谓的现实 bug 一个个修完，他还能剩下多少东西？苏念的脸？王海骂人的声音？还是他为什么一定要活下去？

林川低头看着自己的指腹。那杯水的温度没了，可杯把朝右的画面还在，像一张被撕掉一角的合照。

世界动了。

刘成的手继续向前抓来，林川侧身一躲，那只手擦着他的胳膊过去。风重新灌进楼道，声控灯闪了两下，两个年轻人的鞋尖终于落地。老马手里的欠款单“哗啦”一声抖开，一切都恢复正常，快得像刚才那几秒只是他脑子里的幻觉。

可林川低头，看见自己手背上多了一串细小的数字。

7。

像纹身，也像判决。

刘成皱眉：“你躲什么？”

林川盯着那个数字看了两秒，忽然觉得自己像刚从水里被拽上来，还没喘匀，就又被塞了一把刀。他没有底气，没有计划，更没有三百万。可刚才那几秒静止里，他做了一个很蠢也很真的决定——他不能就这么被拖走。

“刘哥。”林川抬起头，声音沙哑，却比刚才稳，“给我七天。”

刘成看着他。

这一次，他没有立刻嘲笑。他低头看了一眼催收函，目光在某个位置停了半秒。林川顺着他的视线看过去，纸面最下方不知什么时候多了一行极淡的灰字，淡得几乎像印刷错误：展期七日；执行人，刘成。

刘成的眼神冷了一瞬。

那不是惊讶，更不像怜悯。那一瞬间，他看起来不像催债的人，倒像一个突然收到新命令的执行者。

“七天。”刘成慢慢把催收函折回去，“不是我给你的，是账给你的。”

林川喉咙发紧：“什么意思？”

刘成没有回答。他只是整理了一下袖口，动作很轻，像把刚才那点异样也一并整理回衣服里。

“七天后，你要是还拿不出钱，就不是跟我们走一趟这么简单。”他说，“林川，别再用‘想办法’这三个字糊弄账。账听得见，也记得住。”

最后四个字落下来，楼道里莫名冷了一下。

刘成摆了摆手，带人往楼下走。两个年轻人跟上去，老马慢了半拍。临走前，他飞快看了林川一眼。那一眼很复杂，像害怕，又像想提醒什么，可刘成脚步一停，他立刻低下头，夹着欠款单匆匆跟下去。

脚步声顺着楼道往下，一层一层远去。

世界终于又剩下林川一个人。

他靠在门框上，很久才把那口气吐出来。

七天，三百万，怎么还？

这个问题刚从脑子里冒出来，手背上的数字忽然发烫。林川疼得皱眉，低头时，那串“7”像刚被火烧过一样泛出冷白。下一秒，手机屏幕自己亮了。

不是来电。

是一张没有来源的账单：新债目生成；七日内偿还三百万；当前缺口三百万；可校准因果点，城南立交桥，05:17；预结算收益，现金五十万元；释放条件，阻断一次连锁死亡。风险提示写在最后：旧项目关系链将被计入抵押。

林川盯着“五十万”，心跳猛地快了一下。五十万以前对他来说，只是项目预算里一串不算大的数字；现在却像一根插进泥地里的木桩，让他知道这条莫名其妙的路，真的能往前走。

但那不是奖励。更像账先把价码摆出来，冷冰冰地告诉他：你要拿这笔钱，就去拿命验收。林川的拇指压在旧钢笔缺口上，压到疼了，才把视线从“五十万”挪到最后一行。

他的视线很快落到最后一行。

旧项目关系链。

这几个字像一只手，突然伸进胸口，把那些他拼命不敢碰的名字一个个拽出来：苏念，王海，白昼科技，那支没卖掉的旧钢笔，还有一杯他已经记不起温度的水。

屏幕底部又闪出一行更小的字：抵押对象读取中。苏念，有效。王海，错误。王海，查无此人。

林川的手猛地收紧，手机边缘硌得掌心生疼。

他明明记得王海。记得他嘴毒，记得他抽烟，记得他把键盘拍得啪啪响，记得他骂自己“别半路死”。还记得最后一次通宵，王海把那桶红烧牛肉面推给他，嘴上骂“吃，别他妈低血糖死我工位旁边”，转头自己灌了半瓶凉水。

这样一个人，怎么会被写成查无此人？

楼道声控灯终于灭了，黑暗重新压下来。林川站在门口，背后是欠债三百万的出租屋，手里是通往五十万的坐标，脑子里却只剩下那四个字。

查无此人。

两分钟后，他抓起桌上的旧钢笔和手机，冲进凌晨的冷风里。钢笔被他攥在掌心，笔帽缺口硌着肉，疼得很清楚。那是现在唯一好的事——疼还在，说明这东西还没被账收走。

他不是去拿钱。

他是去城南立交桥验账，去 05:17 之前阻断那次连锁死亡，去确认这张账到底会把谁写成代价。

如果修一个 bug 会一点点擦掉苏念，擦掉王海，擦掉他们曾经一起熬过的夜、吃过的泡面、签过的债、说过的狠话——那这七天，他不是在还钱。

他是在跟这张账抢人。

抢一个还没被完全删干净的王海。

也抢一个他已经快记不起温度的苏念。

可他冲下楼时，脑子里忽然冒出一个更冷的问题。

如果王海已经查无此人——

那他现在记得的王海，到底是兄弟，还是一个没删干净的 bug？
