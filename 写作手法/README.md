# 写作手法资料夹

用途：沉淀番茄巅峰榜前十作品的高层写作方法，并在此基础上孵化原创题材与短章试写。

## 资料边界

- 允许：使用用户已登录/VIP 账号进行只读阅读；记录标题结构、开篇机制、升级节奏、爽点兑现、章末钩子、读者情绪触发等高层技法。
- 禁止：保存原书全文、搬运长段落、复刻人物/设定/桥段、输出可替代原书阅读的详细复述。
- 账号操作边界：只读页面，不收藏、不评论、不打赏、不发布、不购买额外内容。
- 完成状态边界：未逐章读完的作品不得标记为“完整精读完成”。

## 单一执行入口

- `00_新书最终写作标准_全量全息版.md` 是唯一写作、返修、审稿和发布准备标准。
- 本目录下其它历史精读、矩阵、题材筛选或试写资料只作原始研究存档，不得作为执行标准读取。
- 已删除 `05_新书工程化台账/` 和《我欠三百万，修bug续命/追踪/》中的冗余标准文件；连续性、规则、声线、反馈、版权、简介、验收和返修口径均已并入总标准。

## 执行规则

写或改《我欠三百万，修bug续命》正文前，只读 `00_新书最终写作标准_全量全息版.md` 和相关正文/细纲；不要再从历史资料中拼接第二套标准。
